## Detailed Description

LibrAA Translator — Japanese AA (Ascii Art) in-page translator

Browse Japanese AA art works (やる夫 series, 2ch/5ch threads) and translate dialogue directly on the page without leaving the site.

### How it works
1. Click the LibrAA icon to activate the toolbar
2. Click "범위 선택" to select an area on the page
3. Japanese text blocks are automatically detected and highlighted
4. Click "전체 번역" to translate all detected blocks, or click individual blocks

### Features
• Block-based translation — only meaningful Japanese text is translated, ASCII art is preserved
• Click translated text to toggle between original and translated
• Multiple LLM providers: OpenAI, Google Gemini, Anthropic Claude, Ollama (local/free)
• Concurrent translation with configurable parallelism
• Customizable highlight color for translated text
• Translation cancel support
• Inline settings — no separate options page needed

### Supported Languages
Translates Japanese to Korean, English, or Japanese (for learning).

### Privacy
Only the text you choose to translate is sent to the LLM API you configure. No browsing data is collected. API keys are stored locally.
